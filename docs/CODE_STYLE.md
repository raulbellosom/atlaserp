# Code Style

- JavaScript solamente, no TypeScript por ahora.
- Componentes React funcionales.
- TailwindCSS para estilos.
- Evitar lógica empresarial en componentes.
- Zod para validaciones compartidas.
- Servicios/API separados de UI.
- No usar emojis en UI o documentación técnica.
- Todo módulo nuevo debe incluir manifest, permisos y documentación.
